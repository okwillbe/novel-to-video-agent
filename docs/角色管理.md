# 全局角色管理设计文档

## 1. 概述

全局角色是用户级别的资产，存储于资产中心，可复用于多个项目。

| 特性 | 说明 |
|------|------|
| 归属级别 | 用户级，非项目级 |
| 主要功能 | 角色设定 + 多形象管理 + 音色绑定 |
| 复用机制 | 复制到项目（非引用） |
| AI 支持 | 从参考图生成、AI 设计形象、局部修改 |

## 2. 数据模型

### 2.1 核心模型 - GlobalCharacter

```java
@Data
@Accessors(chain = true)
@TableName(value = "global_characters", autoResultMap = true)
public class GlobalCharacter {

    @TableId(type = IdType.AUTO)
    private Long id;

    /**
     * UUID，业务主键
     */
    private String characterId;

    /**
     * 所属用户
     */
    private String userId;

    /**
     * 可选：归属文件夹
     */
    private String folderId;

    // ========== 基本信息 ==========

    /**
     * 角色名（必填）
     */
    private String name;

    /**
     * 别名（JSON数组）
     */
    @TableField(typeHandler = JacksonTypeHandler.class)
    private List<String> aliases;

    /**
     * 角色介绍
     */
    private String introduction;

    // ========== 档案 ==========

    /**
     * 档案数据（JSON）
     */
    @TableField(typeHandler = JacksonTypeHandler.class)
    private Object profileData;

    /**
     * 档案是否已确认
     */
    private Integer profileConfirmed;

    // ========== 音色 ==========

    /**
     * qwen-tts 音色ID
     */
    private String voiceId;

    /**
     * 音色类型: qwen-designed | custom
     */
    private String voiceType;

    /**
     * 上传音频URL
     */
    private String customVoiceUrl;

    /**
     * 上传音频媒体ID
     */
    private String customVoiceMediaId;

    /**
     * 绑定的全局音色ID
     */
    private String globalVoiceId;

    // ========== 时间戳 ==========

    @TableField(fill = FieldFill.INSERT)
    private LocalDateTime createdAt;

    @TableField(fill = FieldFill.INSERT_UPDATE)
    private LocalDateTime updatedAt;
}
```

## 6. 角色形象机制

### 6.1 多形象设计

```
角色: 张三
├── 形象 0 (appearanceIndex=0) - 默认形象
│   ├── changeReason: "初始形象"
│   ├── imageUrls: [图1, 图2, 图3, 图4]
│   └── selectedIndex: 2         ← 当前显示 图3
│
├── 形象 1 (appearanceIndex=1) - 便装
│   ├── changeReason: "便装"
│   ├── imageUrls: [图A, 图B]
│   └── selectedIndex: 0
│
└── 形象 2 (appearanceIndex=2) - 战甲
    └── changeReason: "战斗状态"
```

### 6.2 形象字段说明

| 字段 | 说明 |
|------|------|
| appearanceIndex | 形象序号，从 0 开始 |
| changeReason | 形象说明文字（如"便装"、"战甲"） |
| artStyle | 该形象的艺术风格 |
| description | 当前使用的描述词 |
| descriptions | 历史描述词列表（用于切换） |
| imageUrls | 生成的候选图片列表 |
| selectedIndex | 当前选中的图片索引 |

### 6.3 撤销机制

修改图片时：
1. 保存当前状态到 previous* 字段
   - previousImageUrl
   - previousImageUrls
   - previousDescriptions
2. 执行修改
3. 用户可调用 undo-image 撤销

```java
public void savePreviousState(GlobalCharacterAppearance appearance) {
    appearance.setPreviousImageUrl(appearance.getImageUrl());
    appearance.setPreviousImageMediaId(appearance.getImageMediaId());
    appearance.setPreviousImageUrls(appearance.getImageUrls());
    appearance.setPreviousDescriptions(appearance.getDescriptions());
}

public void undoImageChange(String appearanceId) {
    GlobalCharacterAppearance appearance = appearanceMapper.selectById(appearanceId);
    appearance.setImageUrl(appearance.getPreviousImageUrl());
    appearance.setImageMediaId(appearance.getPreviousImageMediaId());
    appearance.setImageUrls(appearance.getPreviousImageUrls());
    appearance.setDescriptions(appearance.getPreviousDescriptions());
    // 清空 previous 字段
    appearance.setPreviousImageUrl(null);
    appearance.setPreviousImageUrls(null);
    appearance.setPreviousDescriptions(null);
    appearanceMapper.updateById(appearance);
}
```

```

    ├── 直接生成模式
    │       │
    │       ▼
    │   POST /api/v1/assets/characters/reference-to-character
    │       │
    │       ▼
    │   创建生成任务 → 返回多张候选图
    │
    └── 提取描述词模式
            │
            ▼
        POST /api/v1/assets/characters/reference-to-character (extract=true)
            │
            ▼
        AI 分析参考图 → 返回描述词
            │
            ▼
        用户编辑描述词 → 提交生成
```

```

    ▼

    │
    ▼
AI 生成详细描述词
    │
    ▼
用户编辑确认 → POST /api/v1/assets/characters/generate-image
    │
    ▼
```


### 8.2 音色清理

删除角色时自动清理无引用的百炼音色：

```java
@Transactional
public void deleteCharacter(String characterId, String userId) {
    GlobalCharacter character = getCharacter(characterId);
    
    // 1. 收集需要清理的音色ID
    List<String> candidateVoiceIds = new ArrayList<>();
    if (character.getVoiceId() != null && "qwen-designed".equals(character.getVoiceType())) {
        candidateVoiceIds.add(character.getVoiceId());
    }
    
    // 2. 删除角色和形象
    deleteAppearances(characterId);
    characterMapper.deleteByCharacterId(characterId, userId);
    
    // 3. 清理无引用的百炼音色
    for (String voiceId : candidateVoiceIds) {
        if (!isVoiceReferencedElsewhere(voiceId, userId, characterId)) {
            bailianVoiceService.deleteVoice(voiceId);
        }
    }
}
```

### 9.2 SSE 推送

```java
// SSE 事件类型
public enum SSEEventType {
    TASK_PROGRESS,      // 任务进度
    IMAGE_GENERATED,    // 图片生成完成
    TASK_ERROR,         // 任务错误
}

// SSE 推送服务
@Service
public class SSEService {
    
    public void pushImageGenerated(String userId, String characterId, List<String> imageUrls) {
        SSEEvent event = new SSEEvent()
            .setType(SSEEventType.IMAGE_GENERATED)
            .setData(Map.of(
                "characterId", characterId,
                "imageUrls", imageUrls
            ));
        pushToUser(userId, event);
    }
}
```

## 12. Service 层实现

```java
@Slf4j
@Service
@RequiredArgsConstructor
public class CharacterService {

    private final GlobalCharacterMapper characterMapper;
    private final GlobalCharacterAppearanceMapper appearanceMapper;
    private final TaskService taskService;
    private final MinioService minioService;
    private final AIGenerateService aiGenerateService;

    /**
     * 创建角色
     */
    @Transactional
    public GlobalCharacter createCharacter(String userId, CreateCharacterRequest request) {
        // 1. 创建角色
        GlobalCharacter character = new GlobalCharacter()
            .setCharacterId(UUID.randomUUID().toString())
            .setUserId(userId)
            .setName(request.getName())
            .setFolderId(request.getFolderId())
            .setIntroduction(request.getDescription())
            .setProfileConfirmed(0);
        characterMapper.insert(character);
        
        // 2. 创建默认形象
        GlobalCharacterAppearance appearance = new GlobalCharacterAppearance()
            .setAppearanceId(UUID.randomUUID().toString())
            .setCharacterId(character.getCharacterId())
            .setAppearanceIndex(0)
            .setChangeReason("初始形象")
            .setArtStyle(request.getArtStyle());
        appearanceMapper.insert(appearance);
        
        // 3. 如果有参考图，触发 AI 生成任务
        if (request.getGenerateFromReference() && 
            CollectionUtils.isNotEmpty(request.getReferenceImageUrls())) {
            triggerImageGeneration(character.getCharacterId(), 0, request);
        }
        
        return character;
    }

    /**
     * 生成角色图片
     */
    public String generateImage(GenerateImageRequest request) {
        GlobalCharacter character = getCharacter(request.getId());
        GlobalCharacterAppearance appearance = getOrCreateAppearance(
            character.getCharacterId(), 
            request.getAppearanceIndex()
        );
        
        // 保存当前状态（支持撤销）
        savePreviousState(appearance);
        
        // 创建异步任务
        Task task = taskService.createTask(Task.builder()
            .taskType(TaskType.ASSET_HUB_IMAGE)
            .targetType(TaskTargetType.GLOBAL_CHARACTER)
            .targetId(character.getCharacterId())
            .inputData(Map.of(
                "appearanceIndex", request.getAppearanceIndex(),
                "artStyle", request.getArtStyle() != null ? request.getArtStyle() : appearance.getArtStyle(),
                "description", appearance.getDescription(),
                "count", request.getCount() != null ? request.getCount() : 4
            ))
            .build());
        
        return task.getTaskId();
    }

    /**
     * 选择候选图片
     */
    @Transactional
    public void selectImage(String appearanceId, Integer imageIndex) {
        GlobalCharacterAppearance appearance = appearanceMapper.selectById(appearanceId);
        
        if (appearance.getImageUrls() == null || imageIndex >= appearance.getImageUrls().size()) {
            throw new IllegalArgumentException("Invalid image index");
        }
        
        // 保存当前状态
        savePreviousState(appearance);
        
        // 更新选中索引和当前图片
        appearance.setSelectedIndex(imageIndex);
        appearance.setImageUrl(appearance.getImageUrls().get(imageIndex));
        appearanceMapper.updateById(appearance);
    }

    /**
     * 撤销图片修改
     */
    @Transactional
    public void undoImageChange(String appearanceId) {
        GlobalCharacterAppearance appearance = appearanceMapper.selectById(appearanceId);
        
        if (appearance.getPreviousImageUrls() == null) {
            throw new IllegalStateException("No previous state to undo");
        }
        
        // 恢复之前的状态
        appearance.setImageUrl(appearance.getPreviousImageUrl());
        appearance.setImageMediaId(appearance.getPreviousImageMediaId());
        appearance.setImageUrls(appearance.getPreviousImageUrls());
        appearance.setDescriptions(appearance.getPreviousDescriptions());
        
        // 清空 previous 字段
        appearance.setPreviousImageUrl(null);
        appearance.setPreviousImageMediaId(null);
        appearance.setPreviousImageUrls(null);
        appearance.setPreviousDescriptions(null);
        
        appearanceMapper.updateById(appearance);
    }
}
```
