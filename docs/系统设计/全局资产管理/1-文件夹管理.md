# 文件夹管理

## API 接口

```
├── 📂 文件夹管理
│   ├── GET    /folders                     获取文件夹列表
│   ├── POST   /folders                     创建文件夹
│   ├── PATCH  /folders/[folderId]          更新文件夹
│   └── DELETE /folders/[folderId]          删除文件夹
```

---

## 1. 概述

文件夹用于组织全局资产中心的角色、场景、音色，实现资产的分类管理。

| 特性 | 说明 |
|------|------|
| 作用范围 | 仅全局资产中心，项目内无文件夹 |
| 层级 | 一层，不支持嵌套 |
| 资产类型 | 角色、场景、音色 |
| 可选性 | 资产可以不归属任何文件夹（未分类） |

## 3. 数据关系

```
User
  │
  └── GlobalAssetFolder (1:N)
        │
        ├── GlobalCharacter (folderId 可选)
        ├── GlobalLocation (folderId 可选)
        └── GlobalVoice (folderId 可选)
```

**特点：**
- 文件夹属于用户，一个用户可有多个文件夹
- 资产通过 `folder_id` 关联文件夹
- `folder_id = null` 表示未分类资产
- 删除文件夹时，资产移至未分类（不会删除资产）

## 5. Service 层

```java
@Service
@Transactional
public class GlobalAssetFolderService {

    private final GlobalAssetFolderRepository folderRepository;
    private final GlobalCharacterRepository characterRepository;
    private final GlobalLocationRepository locationRepository;
    private final GlobalVoiceRepository voiceRepository;

    /**
     * 获取用户所有文件夹
     */
    public List<FolderDto> getFolders(String userId) {
        return folderRepository.findByUserId(userId)
            .stream()
            .map(FolderDto::from)
            .toList();
    }

    /**
     * 创建文件夹
     */
    public FolderDto createFolder(String userId, String name) {
        if (name == null || name.isBlank()) {
            throw new IllegalArgumentException("文件夹名称不能为空");
        }

        GlobalAssetFolder folder = new GlobalAssetFolder();
        folder.setUserId(userId);
        folder.setName(name);

        return FolderDto.from(folderRepository.save(folder));
    }

    /**
     * 更新文件夹名称
     */
    public void updateFolder(String userId, String folderId, String name) {
        GlobalAssetFolder folder = folderRepository.findByIdAndUserId(folderId, userId)
            .orElseThrow(() -> new NotFoundException("文件夹不存在"));

        if (name == null || name.isBlank()) {
            throw new IllegalArgumentException("文件夹名称不能为空");
        }

        folder.setName(name);
        folderRepository.save(folder);
    }

    /**
     * 删除文件夹（资产移至未分类）
     */
    public void deleteFolder(String userId, String folderId) {
        GlobalAssetFolder folder = folderRepository.findByIdAndUserId(folderId, userId)
            .orElseThrow(() -> new NotFoundException("文件夹不存在"));

        // 将关联资产移至未分类
        characterRepository.detachFromFolder(folderId);
        locationRepository.detachFromFolder(folderId);
        voiceRepository.detachFromFolder(folderId);

        // 删除文件夹
        folderRepository.delete(folder);
    }
}
```

## 6. Controller 层

```java
@RestController
@RequestMapping("/api/asset-hub/folders")
public class GlobalAssetFolderController {

    private final GlobalAssetFolderService folderService;

    /**
     * 获取文件夹列表
     */
    @GetMapping
    public ResponseEntity<FolderListResponse> getFolders() {
        String userId = getCurrentUserId(); // 单用户模式可返回固定值
        List<FolderDto> folders = folderService.getFolders(userId);
        return ResponseEntity.ok(new FolderListResponse(folders));
    }

    /**
     * 创建文件夹
     */
    @PostMapping
    public ResponseEntity<FolderResponse> createFolder(@RequestBody @Valid CreateFolderRequest request) {
        String userId = getCurrentUserId();
        FolderDto folder = folderService.createFolder(userId, request.getName());
        return ResponseEntity.ok(new FolderResponse(true, folder));
    }

    /**
     * 更新文件夹
     */
    @PatchMapping("/{folderId}")
    public ResponseEntity<SuccessResponse> updateFolder(
            @PathVariable String folderId,
            @RequestBody @Valid UpdateFolderRequest request) {
        String userId = getCurrentUserId();
        folderService.updateFolder(userId, folderId, request.getName());
        return ResponseEntity.ok(new SuccessResponse(true));
    }

    /**
     * 删除文件夹
     */
    @DeleteMapping("/{folderId}")
    public ResponseEntity<SuccessResponse> deleteFolder(@PathVariable String folderId) {
        String userId = getCurrentUserId();
        folderService.deleteFolder(userId, folderId);
        return ResponseEntity.ok(new SuccessResponse(true));
    }
}
```

## 7. DTO 定义

```java
// 文件夹 DTO
public record FolderDto(
    String id,
    String name
) {
    public static FolderDto from(GlobalAssetFolder folder) {
        return new FolderDto(folder.getId(), folder.getName());
    }
}

// 创建文件夹请求
public record CreateFolderRequest(
    @NotBlank(message = "名称不能为空")
    String name
) {}

// 更新文件夹请求
public record UpdateFolderRequest(
    @NotBlank(message = "名称不能为空")
    String name
) {}

// 响应
public record FolderListResponse(List<FolderDto> folders) {}
public record FolderResponse(boolean success, FolderDto folder) {}
public record SuccessResponse(boolean success) {}
```

## 8. 资产按文件夹筛选

获取资产时支持 `folderId` 参数筛选：

```java
// CharacterController
@GetMapping
public ResponseEntity<?> getCharacters(
        @RequestParam(required = false) String folderId) {
    String userId = getCurrentUserId();

    if (folderId == null) {
        // 获取所有角色
        return ResponseEntity.ok(characterService.getAll(userId));
    } else if ("null".equals(folderId)) {
        // 获取未分类角色
        return ResponseEntity.ok(characterService.getUnclassified(userId));
    } else {
        // 获取指定文件夹内角色
        return ResponseEntity.ok(characterService.getByFolder(userId, folderId));
    }
}
```

## 9. 业务规则

| 规则 | 说明 |
|------|------|
| 命名必填 | 文件夹名称不能为空 |
| 命名唯一性 | 未强制校验（可重名） |
| 删除行为 | 删除文件夹 → 资产移至未分类，不删除资产 |
| 权限隔离 | 用户只能操作自己创建的文件夹 |
| 层级限制 | 不支持嵌套子文件夹 |

## 11. 扩展建议

| 限制 | 扩展方案 |
|------|---------|
| 不支持嵌套 | 添加 `parentId` 字段，实现树形结构 |
| 无排序功能 | 添加 `order` 字段，支持拖拽排序 |
| 无命名校验 | 后端校验名称唯一性 |
| 无批量操作 | 支持批量移动资产到文件夹 |
| 无文件夹颜色/图标 | 添加视觉标识字段 |

## 12. 流程图

### 创建文件夹

```
创建文件夹
    │
    ▼
输入名称 → 校验非空 → 创建记录 → 刷新列表
    │
    ▼
返回成功
```

### 删除文件夹

```
删除文件夹
    │
    ▼
确认提示 → 将关联资产的 folderId 置 null → 删除文件夹 → 刷新列表
    │
    ▼
返回成功
```

### 筛选资产

```
筛选资产
    │
    ├── folderId = null → 显示全部
    ├── folderId = "null" → 显示未分类
    └── folderId = 具体ID → 显示该文件夹内
```

| API 设计 | RESTful CRUD |
