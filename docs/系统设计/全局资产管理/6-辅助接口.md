# 辅助接口设计文档

## 1. 接口总览

| 接口 | 类型 | 功能 |
|------|------|------|
| upload-image | 上传 | 上传用户自定义图片到资产 |
| upload-temp | 上传 | 上传临时文件（base64），返回签名 URL |
| select-image | 选择 | 选择/确认候选图片 |
| undo-image | 撤销 | 撤回到上一版本图片 |

| update-asset-label | 标签更新 | 更新图片黑边标签（重命名时） |
| character-voice | 音色绑定 | 上传/绑定角色音色 |
| voice-design | AI 设计 | AI 设计音色 |
| picker | 选择器 | 获取可选资产列表（用于项目复制） |

### 2.1 upload-image（上传图片）

**REST API:**

```

Content-Type: multipart/form-data
```

**Request:**

```java
    private String labelText;         // 黑边标签文字
}
```

**Response:**

```java
@Data
@AllArgsConstructor
public class UploadImageResponse {
    private String imageKey;    // MinIO 存储路径
    private Integer imageIndex; // 图片索引
    private String imageUrl;   // 访问 URL
```

**Service 实现:**

```java
@Service
@RequiredArgsConstructor
public class ImageUploadService {

    private final MinioService minioService;
    private final CharacterService characterService;
    private final LocationService locationService;

    @Transactional
    public UploadImageResponse uploadImage(String userId, MultipartFile file, UploadImageRequest request) {
        try {
            // 1. 读取图片
            BufferedImage originalImage = ImageIO.read(file.getInputStream());
            int width = originalImage.getWidth();
            int height = originalImage.getHeight();
            
            // 2. 计算标签尺寸
            int fontSize = (int) Math.floor(height * 0.04);
            int padding = (int) Math.floor(fontSize * 0.5);
            int barHeight = fontSize + padding * 2;
            
            // 3. 添加黑边标签
            BufferedImage labeledImage = addLabelBar(originalImage, width, height, barHeight, 
                    request.getLabelText(), fontSize, padding);
            
            // 4. 转换为字节数组
            ByteArrayOutputStream out = new ByteArrayOutputStream();
            ImageIO.write(labeledImage, "jpg", out);
            byte[] imageData = out.toByteArray();
            
            // 5. 上传到 MinIO
            String path = String.format("assets/%s/%s/%d_%d.jpg", 
                    userId, request.getType(), System.currentTimeMillis(), UUID.randomUUID().hashCode());
            String imageUrl = minioService.upload(imageData, path);
            
            // 6. 更新数据库
            updateAssetImageUrls(request, imageUrl);
            
            return new UploadImageResponse(path, request.getImageIndex(), imageUrl);
            
        } catch (IOException e) {
            throw new RuntimeException("Failed to process image", e);
        }
    }

    /**
     * 添加黑边标签
     */
    private BufferedImage addLabelBar(BufferedImage original, int width, int height, 
            int barHeight, String labelText, int fontSize, int padding) {
        
        // 创建带黑边的新图片
        BufferedImage labeled = new BufferedImage(width, height + barHeight, BufferedImage.TYPE_INT_RGB);
        Graphics2D g = labeled.createGraphics();
        
        // 设置抗锯齿
        g.setRenderingHint(RenderingHints.KEY_ANTIALIASING, RenderingHints.VALUE_ANTIALIAS_ON);
        
        // 绘制原图
        g.drawImage(original, 0, barHeight, null);
        
        // 绘制黑边
        g.setColor(Color.BLACK);
        g.fillRect(0, 0, width, barHeight);
        
        // 绘制标签文字
        g.setColor(Color.WHITE);
        g.setFont(new Font("Microsoft YaHei", Font.PLAIN, fontSize));
        g.drawString(labelText, padding, barHeight - padding);
        
        g.dispose();
        return labeled;
    }

    /**
     * 更新资产图片 URL
     */
    private void updateAssetImageUrls(UploadImageRequest request, String imageUrl) {
        if ("character".equals(request.getType())) {
            GlobalCharacterAppearance appearance = characterService.getAppearanceByIndex(
                    request.getId(), 
                    request.getAppearanceIndex() != null ? request.getAppearanceIndex() : 0);
            
            // 保存历史版本
            appearance.setPreviousImageUrls(appearance.getImageUrls());
            
            // 更新图片列表
            List<String> urls = new ArrayList<>(appearance.getImageUrls() != null ? appearance.getImageUrls() : List.of());
            int index = request.getImageIndex() != null ? request.getImageIndex() : urls.size();
            if (index < urls.size()) {
                urls.set(index, imageUrl);
            } else {
                urls.add(imageUrl);
            }
            
            appearance.setImageUrls(urls);
            appearance.setSelectedIndex(index);
            characterService.updateAppearance(appearance);
            
        } else {
            GlobalLocation location = locationService.getLocation(request.getId());
            
            // 更新图片
            GlobalLocationImage image = location.getImages().get(
                    request.getImageIndex() != null ? request.getImageIndex() : 0);
            
            image.setPreviousImageUrl(image.getImageUrl());
            image.setImageUrl(imageUrl);
            locationService.updateLocationImage(image);
        }
    }
}
```

**Service 实现:**

```java
@Service
@RequiredArgsConstructor
public class TempUploadService {

    private final MinioService minioService;

    public UploadTempResponse uploadTemp(String userId, UploadTempRequest request) {
        byte[] data;
        String extension;
        
        if (request.getImageBase64() != null) {
            // 图片模式：从 data URL 解析
            String base64 = request.getImageBase64();
            
            // 匹配 data URL 格式: data:image/png;base64,xxx
            Pattern pattern = Pattern.compile("^data:image/(\\w+);base64,(.+)$");
            Matcher matcher = pattern.matcher(base64);
            
            if (!matcher.matches()) {
                throw new IllegalArgumentException("Invalid image base64 format");
            }
            
            extension = matcher.group(1);
            String base64Data = matcher.group(2);
            data = Base64.getDecoder().decode(base64Data);
            
        } else if (request.getBase64() != null) {
            // 通用模式
            if (request.getExtension() == null) {
                throw new IllegalArgumentException("Extension is required for non-image files");
            }
            extension = request.getExtension();
            data = Base64.getDecoder().decode(request.getBase64());
            
        } else {
            throw new IllegalArgumentException("No data provided");
        }
        
        // 生成唯一路径
        String key = String.format("temp/%s/%d_%s.%s", 
                userId, System.currentTimeMillis(), UUID.randomUUID().toString().substring(0, 8), extension);
        
        // 上传到 MinIO
        minioService.upload(data, key);
        
        // 生成签名 URL（1小时有效）
        String signedUrl = minioService.getSignedUrl(key, 3600);
        
        return new UploadTempResponse(signedUrl, key);
    }
}
```

### 3.1 select-image（选择候选图片）


**Request DTO:**

```java
@Data
public class SelectImageRequest {
    
    @NotBlank
    private String type;  // "character" | "location"
    
    @NotBlank
    private String id;    // 资产ID
    
    private Integer appearanceIndex;  // 仅角色
    
    @NotNull
    private Integer imageIndex;      // 选中的图片索引
    
    private Boolean confirm;          // 是否确认（删除其他候选）
}
```

**Service 实现:**

```java
@Service
@RequiredArgsConstructor
public class ImageSelectService {

    private final CharacterService characterService;
    private final LocationService locationService;
    private final MinioService minioService;

    @Transactional
    public void selectImage(String userId, SelectImageRequest request) {
        if (Boolean.TRUE.equals(request.getConfirm())) {
            confirmSelection(userId, request);
        } else {
            simpleSelection(request);
        }
    }

    /**
     * 确认选择（删除其他候选）
     */
    private void confirmSelection(String userId, SelectImageRequest request) {
        if ("character".equals(request.getType())) {
            GlobalCharacterAppearance appearance = characterService.getAppearanceByIndex(
                    request.getId(), 
                    request.getAppearanceIndex() != null ? request.getAppearanceIndex() : 0);
            
            List<String> imageUrls = appearance.getImageUrls();
            int selectedIndex = request.getImageIndex();
            String selectedUrl = imageUrls.get(selectedIndex);
            String selectedDescription = appearance.getDescriptions() != null 
                    ? appearance.getDescriptions().get(selectedIndex) 
                    : null;
            
            // 删除未选中的图片
            for (int i = 0; i < imageUrls.size(); i++) {
                if (i != selectedIndex && imageUrls.get(i) != null) {
                    minioService.deleteByUrl(imageUrls.get(i));
                }
            }
            
            // 只保留选中的
            appearance.setImageUrls(List.of(selectedUrl));
            appearance.setSelectedIndex(0);
            appearance.setDescription(selectedDescription);
            appearance.setDescriptions(selectedDescription != null ? List.of(selectedDescription) : null);
            
            characterService.updateAppearance(appearance);
            
        } else {
            GlobalLocation location = locationService.getLocation(request.getId());
            GlobalLocationImage selectedImage = location.getImages().get(request.getImageIndex());
            
            // 删除其他图片
            for (int i = 0; i < location.getImages().size(); i++) {
                if (i != request.getImageIndex()) {
                    minioService.deleteByUrl(location.getImages().get(i).getImageUrl());
                }
            }
            
            location.setImages(List.of(selectedImage));
            locationService.updateLocation(location);
        }
    }

    /**
     * 普通选择（不删除其他候选）
     */
    private void simpleSelection(SelectImageRequest request) {
        if ("character".equals(request.getType())) {
            GlobalCharacterAppearance appearance = characterService.getAppearanceByIndex(
                    request.getId(),
                    request.getAppearanceIndex() != null ? request.getAppearanceIndex() : 0);
            appearance.setSelectedIndex(request.getImageIndex());
            characterService.updateAppearance(appearance);
        } else {
            GlobalLocation location = locationService.getLocation(request.getId());
            for (GlobalLocationImage image : location.getImages()) {
                image.setIsSelected(image.getImageId().equals(
                        location.getImages().get(request.getImageIndex()).getImageId()) ? 1 : 0);
            }
            locationService.updateLocation(location);
        }
    }
}
```

**Service 实现:**

```java
@Service
@RequiredArgsConstructor
public class ImageUndoService {

    private final CharacterService characterService;
    private final LocationService locationService;

    @Transactional
    public void undoImage(UndoImageRequest request) {
        if ("character".equals(request.getType())) {
            undoCharacterImage(request);
        } else {
            undoLocationImage(request);
        }
    }

    private void undoCharacterImage(UndoImageRequest request) {
        GlobalCharacterAppearance appearance = characterService.getAppearanceByIndex(
                request.getId(),
                request.getAppearanceIndex() != null ? request.getAppearanceIndex() : 0);
        
        // 检查是否有历史版本
        if (appearance.getPreviousImageUrls() == null) {
            throw new IllegalStateException("No previous version to undo");
        }
        
        // 恢复上一版本
        appearance.setImageUrls(appearance.getPreviousImageUrls());
        appearance.setImageUrl(appearance.getPreviousImageUrls().get(0));
        appearance.setDescriptions(appearance.getPreviousDescriptions());
        appearance.setDescription(appearance.getPreviousDescriptions() != null 
                ? appearance.getPreviousDescriptions().get(0) 
                : null);
        
        // 清空历史数据
        appearance.setPreviousImageUrls(null);
        appearance.setPreviousDescriptions(null);
        
        characterService.updateAppearance(appearance);
    }

    private void undoLocationImage(UndoImageRequest request) {
        GlobalLocation location = locationService.getLocation(request.getId());
        GlobalLocationImage image = location.getImages().get(0);
        
        if (image.getPreviousImageUrl() == null) {
            throw new IllegalStateException("No previous version to undo");
        }
        
        // 恢复上一版本
        image.setImageUrl(image.getPreviousImageUrl());
        image.setPreviousImageUrl(null);
        
        locationService.updateLocationImage(image);
    }
}
```

**Service 实现:**

```java
@Service
@RequiredArgsConstructor
public class ReferenceToCharacterService {

    private final CharacterService characterService;
    private final TaskService taskService;
    private final AIProviderFactory aiProviderFactory;
    private final MinioService minioService;
    private final SSEService sseService;

    public String referenceToCharacter(String userId, ReferenceToCharacterRequest request) {
        // 创建任务
        Task task = taskService.createTask(Task.builder()
            .taskType(TaskType.ASSET_HUB_IMAGE)
            .targetType(TargetType.GLOBAL_CHARACTER_APPEARANCE)
            .targetId(request.getAppearanceId())
            .userId(userId)
            .inputData(toMap(request))
            .build());
        
        // 提交到队列
        taskService.submitToQueue(task);
        
        return task.getTaskId();
    }

    @Transactional
    public void handleReferenceToCharacterTask(Task task) {
        String userId = task.getUserId();
        ReferenceToCharacterRequest request = fromMap(task.getInputData());
        
        // 提取模式
        if (Boolean.TRUE.equals(request.getExtractOnly())) {
            String description = extractDescription(userId, request.getReferenceImageUrls());
            taskService.completeTask(task.getTaskId());
            sseService.pushTaskResult(userId, task.getTaskId(), Map.of("description", description));
            return;
        }
        
        // 生成模式
        String prompt = buildPrompt(request);
        List<String> generatedUrls = new ArrayList<>();
        
        for (int i = 0; i < request.getCount(); i++) {
            // 更新进度
            int progress = (i + 1) * 100 / request.getCount();
            taskService.updateProgress(task.getTaskId(), progress);
            sseService.pushTaskProgress(userId, task.getTaskId(), progress, "processing");
            
            // 调用图片模型
            AiProvider imageProvider = aiProviderFactory.getProvider("character");
            AiImageResponse response = imageProvider.generateImage(AiImageRequest.builder()
                .prompt(prompt)
                .aspectRatio("3:2")
                .referenceImages(request.getReferenceImageUrls())
                .build());
            
            // 添加标签
            byte[] labeledImage = minioService.addLabelBar(response.getImageData(), 
                    "character", request.getCharacterName());
            
            // 上传到 MinIO
            String url = minioService.upload(labeledImage, 
                    String.format("assets/%s/character/%s_%d.jpg", userId, task.getTaskId(), i));
            
            generatedUrls.add(url);
        }
        
        // 更新数据库
        GlobalCharacterAppearance appearance = characterService.getAppearance(request.getAppearanceId());
        appearance.setImageUrls(generatedUrls);
        appearance.setSelectedIndex(0);
        characterService.updateAppearance(appearance);
        
        // 完成任务
        taskService.completeTask(task.getTaskId());
        sseService.pushImageGenerated(userId, request.getCharacterId(), generatedUrls);
    }

    /**
     * 从参考图提取描述词
     */
    private String extractDescription(String userId, List<String> referenceUrls) {
        AiProvider visionProvider = aiProviderFactory.getProvider("analysis");
        AiCompletion completion = visionProvider.complete(AiCompletionRequest.builder()
            .model(visionProvider.getDefaultModel())
            .messages(List.of(new AiMessage("user", 
                    "请分析这些参考图片，生成角色外观描述。返回简洁的描述文字。",
                    referenceUrls)))
            .build());
        
        return completion.getText();
    }

    /**
     * 构建提示词
     */
    private String buildPrompt(ReferenceToCharacterRequest request) {
        String basePrompt = request.getCustomDescription() != null 
                ? request.getCustomDescription() 
                : CHARACTER_REFERENCE_TO_SHEET;
        
        StringBuilder prompt = new StringBuilder(basePrompt);
        prompt.append("，").append(CHARACTER_PROMPT_SUFFIX);
        
        if (request.getArtStyle() != null) {
            prompt.append("，").append(request.getArtStyle());
        }
        
        return prompt.toString();
    }

    private static final String CHARACTER_REFERENCE_TO_SHEET = 
            "根据参考图生成角色三视图，正面、侧面、背面";
    
    private static final String CHARACTER_PROMPT_SUFFIX = 
            "角色设定图，全身像，正面视角，清晰的面部特征，三视图，高质量，详细细节";
}
```

```java
@Service
@RequiredArgsConstructor
public class VoiceDesignService {

    private final AIProviderFactory aiProviderFactory;
    private final UserConfigService userConfigService;

    /**
     * AI 设计音色
     */
    public VoiceDesignResponse designVoice(String userId, VoiceDesignRequest request) {
        // 1. 参数校验
        validateRequest(request);
        
        // 2. 获取百炼 API Key
        String apiKey = userConfigService.getProviderApiKey(userId, "bailian");
        
        // 3. 调用百炼音色设计 API
        AiProvider ttsProvider = aiProviderFactory.getProvider("bailian-tts");
        VoiceDesignResult result = ttsProvider.designVoice(VoiceDesignRequest.builder()
            .voicePrompt(request.getVoicePrompt())
            .previewText(request.getPreviewText())
            .preferredName(request.getPreferredName())
            .language(request.getLanguage())
            .apiKey(apiKey)
            .build());
        
        // 4. 返回结果
        return new VoiceDesignResponse(
            result.getVoiceId(),
            result.getAudioBase64(),
            result.getTargetModel()
        );
    }

    private void validateRequest(VoiceDesignRequest request) {
        if (request.getVoicePrompt() == null || request.getVoicePrompt().trim().isEmpty()) {
            throw new IllegalArgumentException("Voice prompt is required");
        }
        if (request.getPreviewText() == null || request.getPreviewText().trim().isEmpty()) {
            throw new IllegalArgumentException("Preview text is required");
        }
    }
}
```

**百炼音色设计调用:**

```java
@Slf4j
@Service
@RequiredArgsConstructor
public class BailianTtsProvider implements AiProvider {

    @Override
    public String getProviderType() {
        return "bailian-tts";
    }

    /**
     * 设计音色
     */
    public VoiceDesignResult designVoice(VoiceDesignParams params) {
        try {
            // 调用百炼 qwen-tts-vd API
            HttpRequest request = HttpRequest.newBuilder()
                .uri(URI.create("https://dashscope.aliyuncs.com/api/v1/services/audio/tts/voice-design"))
                .header("Authorization", "Bearer " + params.getApiKey())
                .header("Content-Type", "application/json")
                .POST(HttpRequest.BodyPublishers.ofString(toJson(params)))
                .build();
            
            HttpResponse<String> response = HttpClient.newHttpClient()
                .send(request, HttpResponse.BodyHandlers.ofString());
            
            if (response.statusCode() != 200) {
                throw new RuntimeException("Bailian API error: " + response.body());
            }
            
            Map<String, Object> result = parseJson(response.body());
            
            return VoiceDesignResult.builder()
                .voiceId((String) result.get("voice_id"))
                .audioBase64((String) result.get("audio"))
                .targetModel((String) result.get("target_model"))
                .build();
                
        } catch (Exception e) {
            log.error("Failed to design voice", e);
            throw new RuntimeException("Failed to design voice", e);
        }
    }
}
```

```java
@Service
@RequiredArgsConstructor
public class MinioService {

    /**
     * 更新图片标签
     */
    public String updateLabelBar(String imageUrl, String newLabelText) {
        try {
            // 1. 下载原图片
            byte[] originalData = download(imageUrl);
            BufferedImage original = ImageIO.read(new ByteArrayInputStream(originalData));
            
            int width = original.getWidth();
            int height = original.getHeight();
            
            // 2. 检测黑边高度
            int barHeight = detectLabelBarHeight(original);
            
            // 3. 裁剪掉旧黑边
            BufferedImage cropped = original.getSubimage(0, barHeight, width, height - barHeight);
            
            // 4. 计算新标签尺寸
            int fontSize = (int) Math.floor(cropped.getHeight() * 0.04);
            int padding = (int) Math.floor(fontSize * 0.5);
            int newBarHeight = fontSize + padding * 2;
            
            // 5. 合成新黑边
            BufferedImage labeled = new BufferedImage(width, cropped.getHeight() + newBarHeight, BufferedImage.TYPE_INT_RGB);
            Graphics2D g = labeled.createGraphics();
            
            g.drawImage(cropped, 0, newBarHeight, null);
            g.setColor(Color.BLACK);
            g.fillRect(0, 0, width, newBarHeight);
            g.setColor(Color.WHITE);
            g.setFont(new Font("Microsoft YaHei", Font.PLAIN, fontSize));
            g.drawString(newLabelText, padding, newBarHeight - padding);
            g.dispose();
            
            // 6. 上传为新文件（强制 URL 变化）
            ByteArrayOutputStream out = new ByteArrayOutputStream();
            ImageIO.write(labeled, "jpg", out);
            
            String newPath = generateNewPath(imageUrl, newLabelText);
            return upload(out.toByteArray(), newPath);
            
        } catch (IOException e) {
            throw new RuntimeException("Failed to update label bar", e);
        }
    }

    /**
     * 检测黑边高度
     */
    private int detectLabelBarHeight(BufferedImage image) {
        int height = image.getHeight();
        int width = image.getWidth();
        
        // 从顶部向下检测黑色区域
        for (int y = 0; y < 50; y++) {
            int rgb = image.getRGB(width / 2, y);
            if (rgb != Color.BLACK.getRGB()) {
                return y;
            }
        }
        return 0;
    }

    /**
     * 生成新路径（确保 URL 变化）
     */
    private String generateNewPath(String oldPath, String newName) {
        return oldPath.replace(".jpg", "_" + newName.hashCode() + "_" + System.currentTimeMillis() + ".jpg");
    }
}
```

```java
public class CharacterVoiceService {

    private final CharacterService characterService;
    private final MinioService minioService;

    /**
     * 上传音频绑定
     */
    @Transactional
    public void bindUploadedVoice(String userId, String characterId, MultipartFile file) {
        // 1. 验证文件类型
        validateAudioFile(file);
        
        // 2. 上传到 MinIO
        String audioUrl = minioService.upload(file.getBytes(), 
                "voices/" + userId + "/" + characterId + "/" + System.currentTimeMillis() + ".mp3");
        
        // 3. 更新角色
        GlobalCharacter character = characterService.getCharacter(characterId);
        character.setVoiceType("uploaded");
        character.setVoiceId(null);
        character.setCustomVoiceUrl(audioUrl);
        
        characterService.updateCharacter(character);
    }

    /**
     * AI 设计绑定
     */
    @Transactional
    public void bindDesignedVoice(String characterId, String voiceId, String audioBase64) {
        // 1. 解码音频
        byte[] audioData = Base64.getDecoder().decode(audioBase64);
        
        // 2. 上传预览音频
        String audioUrl = minioService.upload(audioData,
                "voices/ai/" + characterId + "/" + voiceId + ".mp3");
        
        // 3. 更新角色
        GlobalCharacter character = characterService.getCharacter(characterId);
        character.setVoiceType("qwen-designed");
        character.setVoiceId(voiceId);
        character.setCustomVoiceUrl(audioUrl);
        
        characterService.updateCharacter(character);
    }

    private void validateAudioFile(MultipartFile file) {
        String contentType = file.getContentType();
        if (contentType == null || !contentType.startsWith("audio/")) {
            throw new IllegalArgumentException("Invalid audio file type");
        }
    }
}
```

```java
@Service
@RequiredArgsConstructor
public class AssetPickerService {

    private final CharacterService characterService;
    private final LocationService locationService;
    private final VoiceService voiceService;

    public AssetPickerResponse getPicker(String userId, String type) {
        AssetPickerResponse response = new AssetPickerResponse();
        
        if ("character".equals(type) || type == null) {
            response.setCharacters(getCharacterPickerItems(userId));
        }
        
        if ("location".equals(type) || type == null) {
            response.setLocations(getLocationPickerItems(userId));
        }
        
        if ("voice".equals(type) || type == null) {
            response.setVoices(getVoicePickerItems(userId));
        }
        
        return response;
    }

    private List<CharacterPickerItem> getCharacterPickerItems(String userId) {
        List<GlobalCharacter> characters = characterService.listCharacters(userId, null);
        
        return characters.stream()
                .map(c -> new CharacterPickerItem(
                        c.getCharacterId(),
                        c.getName(),
                        c.getFolder() != null ? c.getFolder().getName() : null,
                        getPreviewUrl(c),
                        c.getAppearances() != null ? c.getAppearances().size() : 0,
                        c.getGlobalVoiceId() != null || c.getCustomVoiceUrl() != null
                ))
                .collect(Collectors.toList());
    }

    private String getPreviewUrl(GlobalCharacter character) {
        if (character.getAppearances() == null || character.getAppearations().isEmpty()) {
            return null;
        }
        GlobalCharacterAppearance appearance = character.getAppearances().get(0);
        if (appearance.getImageUrls() == null || appearance.getImageUrls().isEmpty()) {
            return null;
        }
        return appearance.getImageUrls().get(appearance.getSelectedIndex() != null ? appearance.getSelectedIndex() : 0);
    }
}
```

## 7. 总结

| 接口 | 核心原理 | 关键技术 |
|------|----------|----------|
| upload-image | 上传 + 黑边标签 | BufferedImage 图片处理 |
| upload-temp | base64 → 存储 → 签名 URL | 临时文件管理 |
| select-image | 切换 selectedIndex | 候选图管理 |
| undo-image | previous* 字段恢复 | 撤销机制 |
| reference-to-character | AI Vision + 图片生成 | 参考图驱动生成 |
| update-asset-label | 剪切 + 新标签 + 新 URL | 标签更新 |
| character-voice | 音频上传 + 音色绑定 | 多模态支持 |
| voice-design | 百炼 qwen-tts-vd | AI 音色设计 |
| picker | 简化资产列表 | 项目复制支持 |

**设计亮点：**

- 黑边标签：角色名自动叠加，方便识别
- 撤销机制：previous* 字段保存历史版本
- 参考图生成：AI Vision 提取 + AI 图片生成
- 标签更新：生成新 URL 绕过浏览器缓存
- AI 音色设计：百炼 qwen-tts-vd，实时试听
